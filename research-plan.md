# 따릉이 사용 현황 분석 — 연구 계획서

## 1. 문제 상황 (Problem Statement)

서울시 공공자전거 따릉이는 대여소 기반(station-based) 운영 방식으로, **대여소의 위치와 밀도가 사용자 접근성과 직결**된다. 그러나 대여소가 실제 수요에 맞게 배치되어 있는지에 대한 정량적 근거는 부족하다.

핵심 질문:

- 따릉이 대여소의 위치가 **사용자 친화적으로(수요가 있는 곳에 충분히)** 분포해 있는가?
- 수요는 많지만 대여소가 부족한 **공급 부족(under-served) 지역**이 존재하는가?
- 반대로 이용량 대비 대여소가 과잉인 **공급 과잉(over-served) 지역**이 존재하는가?

대여소가 수요와 불일치할 경우 (1) 이용자 불편(가까운 곳에 자전거 없음), (2) 자전거 재배치(rebalancing) 운영비 증가, (3) 신규 대여소 투자 비효율이 발생한다.

## 2. 분석 목적 (Objective)

따릉이 이용 현황을 공간적·시계열적으로 분석하여 다음을 도출한다.

1. 현재 대여소 분포의 **수요-공급 정합성** 진단
2. **신규 대여소 추가 필요 여부** 판단 및 **추천 위치** 제시
3. 향후 수요를 **예측**하여 선제적 배치 의사결정을 지원

성공 기준 (검증 가능한 목표):

- [ ] 전체 대여소를 좌표 기준으로 지도에 매핑 (좌표 결측/이상치 제외율 보고)
- [ ] 대여소별 월간 이용량 시각화 (단계구분도/버블맵)
- [ ] 복수 시계열 모델을 동일 조건으로 백테스트 → 오차(MAPE/RMSE)로 비교 → 최적 모델 선정 근거 보고
- [ ] 선정 모델로 대여소별 향후 N개월 수요 예측
- [ ] "예측 수요 평균 이상 & 대여소 밀도 낮음" 후보 지역 목록 산출
- [ ] 위 결과를 비즈니스 인사이트(추가/이전/유지)로 정리

## 3. 데이터 (Data Sources)

| 데이터셋 | 출처 | 주요 필드 | 갱신 | 용도 |
|---|---|---|---|---|
| 대여소 마스터 정보 (OA-21235, `tbCycleStationInfo`) | 서울 열린데이터광장 OpenAPI | 대여소ID, 주소, 위도, 경도 | 수시 | 대여소 위치(공간) |
| 대여소별 이용정보(월별) (OA-15249) | 서울 열린데이터광장 OpenAPI / CSV | 년월, 대여소번호, 대여소명, 대여건수 | 반기 | 월별 수요(시계열) — 단순·정제된 대여소-월 총량 |
| 이용정보(월별) (OA-15248) | 서울 열린데이터광장 OpenAPI / CSV | 대여일자, 대여소번호, 대여소명, 대여구분(정기/일일), 성별, 연령대, 이용건수, 이동거리, 이동시간, 운동량, 탄소량 | 반기 | 세그먼트별 수요 + 이용 강도/임팩트 피처 |

- 세 데이터셋은 **대여소번호/대여소ID**를 키로 조인한다.
- **OA-15248은 OA-15249의 상위집합**: 대여소 × 성별 × 연령대 × 대여구분 단위라, `(대여소, 년월)`로 집계하면 OA-15249의 대여건수와 동일한 대여소-월 총량을 도출할 수 있다. → OA-15248을 메인으로 쓰되, 총량 검증용으로 OA-15249와 교차 확인.
- API 호출 형식: `http://openapi.seoul.go.kr:8088/{API_KEY}/json/{SERVICE}/{START}/{END}/`
  - 키는 `.env`의 `OPENAPI`에서 로드 (`python-dotenv`).
- 보조(선택) 데이터: 행정동 경계(GeoJSON), 지하철역 위치, 인구/생활인구 — 접근성·수요 맥락 보강용.

### 데이터 한계 (사전 고지)

- 개별 통행(출발지→도착지 OD) 정보는 본 API들에 없음(별도 대여이력 API 필요). OA-15248은 이동거리·시간을 **집계값**으로만 제공.
- 성별·연령대 미상(결측) 구간이 있을 수 있어 세그먼트 분석 시 결측 처리 필요.
- 월별 데이터 갱신이 **반기 주기**라 최신성 지연 가능.
- 대여소는 신설/폐쇄로 시점별 모집단이 변동 → 시계열 결측·관측 길이 차이 처리 필요.

## 4. 분석 방법 (Methodology)

### 단계 1 — 수집 및 전처리
- OpenAPI로 대여소 마스터 + 월별 이용 데이터 수집, CSV 백업.
- 대여소번호 기준 조인. 좌표 유효성 검증(서울 범위 밖 좌표 제거), 중복·결측 처리.
- 시계열 정렬: `(대여소번호, 년월)` 패널 구성, 신설/폐쇄에 따른 결측 명시.

### 단계 2 — 대여소 위치 매핑 (방법 1)
- `folium`/`kepler.gl`로 전체 대여소 좌표 표시 (마커 클러스터).
- 행정동 경계 오버레이로 구역별 대여소 **밀도(개수/면적)** 산출.

### 단계 3 — 이용량 공간 시각화 (방법 2)
- 대여소별 월 평균 대여건수를 **버블 크기/색상**으로 표현.
- 행정동 단위 **단계구분도(choropleth)**로 수요 집중 지역 식별.
- 계절성(월별 패턴) 파악 — 여름 성수기/겨울 비수기 등.

### 단계 4 — 접근성·정합성 진단
- 대여소 **밀도 vs 이용량** 교차 분석: 4분면(고수요-고밀도 / 고수요-저밀도 / 저수요-고밀도 / 저수요-저밀도).
- "고수요-저밀도" = **공급 부족 후보**, "저수요-고밀도" = **공급 과잉 후보**.

### 단계 4.5 — 세그먼트·이용 강도 분석 (OA-15248 활용)
- **인구통계 세그먼트**: 성별·연령대·대여구분(정기/일일)별 이용 패턴 → 지역별 주 이용층 프로파일.
- **이용 강도 지표**: 대여건수당 평균 이동거리·이동시간 산출 → 통근형(짧고 잦음) vs 레저형(길고 드묾) 대여소 분류.
- **임팩트 지표**: 탄소절감량·운동량으로 지역별 사회적 효익 정량화.

### 단계 5 — 시계열 수요 예측: 모델 비교·선정 (방법 3)

여러 예측 모델을 **동일한 조건으로 학습·평가**한 뒤, 테스트 성능으로 최적 모델을 선정한다.

**후보 모델 (난이도 단계별)**
- 베이스라인: 직전값(naive), 계절 naive(작년 동월), 이동평균
- 통계 모델: SARIMA, (지수평활) ETS
- 라이브러리: Prophet
- 머신러닝: 시차/계절 피처 + Gradient Boosting(XGBoost/LightGBM)
- (관측 충분 시) 딥러닝: LSTM 등

**예측 피처 (OA-15248 활용)**
- 시계열 자체 외에, 대여소별 세그먼트 구성(연령대·정기/일일 비율), 평균 이동거리·시간(통근형/레저형 성격)을 머신러닝 모델의 보조 피처로 투입 → 단순 시계열보다 설명력 향상 가능.

**공통 평가 프로토콜 (공정 비교)**
- 시계열 교차검증: rolling/expanding window backtest (미래 누설 방지).
- 평가 지표: MAPE, RMSE, MAE (+ 베이스라인 대비 개선율).
- 동일한 train/test 분할·전처리·예측 구간을 모든 모델에 적용.

**모델 선정**
- 베이스라인 대비 유의미하게 우수한 모델만 채택 (단순 모델 못 이기면 단순 모델 채택).
- 대여소별 특성 차이 클 경우, **전역 단일 모델 vs 대여소별 최적 모델** 둘 다 비교 후 선택.
- 관측 길이 짧은 신설 대여소는 인접 대여소·구역 평균으로 보완.
- 결과물: **모델별 오차 비교표 + 선정 근거**.

### 단계 6 — 미래 수요 분류 (방법 4)
- 예측된 향후 수요를 전체/구역 **평균 대비 이상/이하**로 분류.
- 분류 × 대여소 밀도 결합:
  - 예측 수요 **평균 이상 + 저밀도** → **신규 대여소 추천 지역**
  - 예측 수요 **평균 이하 + 고밀도** → 통폐합/이전 검토 지역

### 단계 7 — 추천 위치 산출
- 공급 부족 후보 구역 내에서 기존 대여소와의 거리·수요 핫스팟을 고려해 신규 위치 후보 도출.

## 5. 결론 및 비즈니스 인사이트 (Business Insight)

분석 결과로부터 다음 의사결정을 지원한다.

1. **신규 대여소 투자 우선순위**
   - "고수요·저공급" 지역을 우선순위로 제시 → 투자 대비 이용 증가 효과 극대화.
2. **자원 재배치(이전·통폐합)**
   - "저수요·고공급" 대여소를 식별 → 자전거·대여소를 수요 지역으로 재배치하여 운영비 절감.
3. **선제적 운영 계획**
   - 시계열 예측으로 성수기 수요 급증 지역에 자전거 재배치(rebalancing) 사전 배치 → 결품률 감소, 이용자 만족도 향상.
4. **접근성 개선의 정량 근거**
   - "사용자 친화적 분포" 질문에 수요-공급 정합도 지표로 정량 답변 → 정책 보고·예산 근거로 활용.
5. **세그먼트 맞춤 운영 (OA-15248 기반)**
   - 지역별 주 이용층(연령대·정기/일일)·이용 성격(통근형 vs 레저형)에 맞춰 대여소 규모·자전거 종류·마케팅 차별화.
6. **사회적 효익 정량화**
   - 탄소절감량·운동량으로 따릉이 사업의 환경·건강 효과를 수치화 → 예산 정당화 및 ESG 보고 근거.

기대 효과: 이용자 접근성 향상, 재배치 운영비 절감, 신규 대여소 투자 효율 개선, 세그먼트 맞춤 운영, 사회적 효익 입증.

## 6. 산출물 (Deliverables)

- 대여소 위치/이용량 인터랙티브 지도
- 수요-공급 4분면 진단 리포트
- 대여소별 수요 예측 결과 + 모델 검증 지표
- 신규 대여소 추천 위치 목록 및 우선순위
